Imágenes y fondos de una página web

Con el auge de Internet surgió el protocolo (HTTP: Protocolo de Transferencia de Hipertexto) que permite la estandarización de los formatos gráficos en la Internet, estableciendo como base los archivos gif (archivo de intercambio gráfico) y jpg (formato de archivo gráfico profesional), ambos poseen la característica de ser de un tamaño pequeño.

Un elemento indispensable en el ambiente de multimedia corresponde a las imágenes, permite engrandecer la apariencia de las páginas web, además de servir como fondo de la página, mejorando notablemente el diseño.

En el presente documento se abarca como insertar una imagen en una página web y los diferentes atributos que la acompañan, mediante etiquetas html y Dreamweaver CS6, además colocar una imagen como fondo de una página web, el procedimiento es elaborado por medio de etiquetas html y hoja de estilos en cascada.

Insertar imágenes en un documento, por medio de etiquetas html

Se explica el funcionamiento de la etiqueta <img>. El atributo más importante es src. El atributo src (source) indica el nombre o URL de la imagen que se desea incluir en línea, escrito entre comillas. La ruta del archivo sigue las reglas del direccionamiento absoluto o relativo. Así, para un archivo jpg llamado 5.9.2.jpg, que se encuentra en el mismo directorio que la página, se usa la siguiente etiqueta:

<img src="5.9.2.jpg">

Sí, el archivo imagen se encuentra en un directorio por encima del actual, la etiqueta es:

<img src="../5.9.2.jpg">
