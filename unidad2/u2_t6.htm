Etiquetas de formato de texto

Editar la información es un proceso importante en el diseño de una página web; conlleva a mejorar la presentación y distribución del texto en la pantalla, en un editor para contenidos en la plataforma de Internet es complejo brindar una apariencia agradable del texto de la página web y se lleva a cabo a través de etiquetas de formato de texto y se tienen las siguientes:

  • Encabezados <H1...H6>
  • Párrafo <P>
  • Salto de línea <br>
  • Lista numerada <ol>
  • Lista viñetas <ul>
  • Tipo de fuentes <font>
  • Texto preformateado <pre>
  • Superíndice, subíndice <sup> <sub>
  • Estilo (negrita, cursiva, subrayado) <b>, <i>, <u>
  • Acentos <acute>
  • Comentarios <!>
  • Líneas horizontales <hr>
  • Sangrías <blockquote>
  • Espacio en blanco <&nbsp>

Usar un editor de páginas web facilita el diseño de un archivo html, hay que seleccionar el texto y dar clic en el icono correspondiente al formato que se desee, hoy en día, existe una gran variedad de iconos que cubren las características que debe cumplir una edición de texto como son fuentes, estilos, alineación, lista numerada y colores para el texto (Castro, 2003, pp. 85 - 95).

Diseño y formato de una página web, empleando las hojas de estilo en cascada (CSS)

Una parte muy importante en la creación de una página web es la aplicación del diseño y formato, existe una técnica que permite realizar esta función de una manera más eficiente cuando se aplica a una gran cantidad de páginas web conocida como hojas de estilos en cascada, es práctico definir una regla que se va a aplicar a 50 archivos html y solamente se tiene que definir una sola vez para posteriormente aplicarse a cada elemento de la página web. La regla que se define se guarda en un archivo con extensión CSS y dentro de él se declaran las clases que tienen propiedades muy específicas. Realizar la aplicación del  diseño y formato a los seis archivos de las  especialidades, empleando las  hojas de  estilo en  cascada, en la vista diseño. Las páginas web a las que se les aplica la técnica CSS son las siguientes:

aeronautica.htm

automotriz.htm

construccion.htm

electricidad.htm

mantenimiento.htm

soldadura.htm
