Publicación en Internet

El objetivo de diseñar un sitio web, es mostrar texto, imágenes, animaciones y videos de un tema en particular, es necesario publicar, el procedimiento para mostrar la información en Internet es el siguiente: crear un sitio web, posteriormente abrir el sitio para comenzar a diseñar cada uno de los archivos htm que se necesitan, mostrar la información al momento que un usuario la solicite, se requiere que el servidor web posea un nombre registrado en Internet o una dirección IP (Protocolo de Internet), el servidor en el ambiente Windows para crear y administrar los servicios se llama Internet Information Server (IIS), este elemento crea la carpeta para colocar en ella los datos que se desean mostrar en la Internet, ubicada en el disco local C:, dentro de la carpeta Inetpub y en el subdirectorio wwwroot, en el cual deben colocarse las páginas que se desean publicar, en este caso la carpeta cecyt72IVE, la cual contiene el sitio web académico, como se observa en la figura 19.
