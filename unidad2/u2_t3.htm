Definir el sitio web

Diseñar y crear páginas web, es una tarea relativamente sencilla cuando se trata de unas cuantas páginas, sin embargo, cuando la cantidad de ellas aumenta considerablemente, es necesario el empleo de un software para facilitar el trabajo y proporcionar la ventaja de definir un sitio para lograr una administración eficiente de los datos de la aplicación que se desarrolle.

En el video 37 se describe el procedimiento para definir un sitio en Dreamweaver para la aplicación web, es necesario crear una carpeta local en el disco duro que lleva el nombre del sitio y es en donde se almacenan las copias de los archivos de trabajo del sitio, en este caso la carpeta local se ubica en el disco duro C: dentro de un subdirectorio llamado Humberto (nombre de la persona que la crea) y dentro de él genera un folder con el nombre de la escuela seguida del grupo, por ejemplo cecyt72IVE. También se tiene que crear una carpeta remota con el nombre de cecyt72IVE, se crea dentro del subdirectorio wwwroot, se localiza dentro de la carpeta de inetpub. Estas dos últimas carpetas son creadas por un programa preinstalado en el sistema, el cual lleva por nombre Servicio de Información en Internet (IIS) es el encargado de publicar la información en Internet, administrar los servicios WWW, dicho software esta dentro del sistema operativo de Windows 10.
