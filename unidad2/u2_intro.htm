Introducción al diseño de páginas web

La Internet, es actualmente una plataforma de desarrollo para una amplia gama de aplicaciones: académicas, comerciales, gubernamentales, militares y entretenimiento.

Internet en 1990, era una red de uso militar y algunas instituciones académicas de grandes dimensiones como la UNAM e IPN, que deseaban mostrar contenidos, mediante textos, las ligas hacia otros sitios eran a través de hipertextos posteriormente en 1995, la red de Internet empieza a mostrar imágenes, aplicaciones multimedia y pequeñas animaciones.

Las empresas comerciales ponen énfasis en el desarrollo de aplicaciones que producen grandes cantidades de dinero, razón por la cual se desarrollan los lenguajes de programación para Internet, primeramente se programó mediante el lenguaje de marcas de hipertexto, utilizando etiquetas del lenguaje HTML y empleando para ello, un editor de textos y los sitios eran montados en pequeños servidores web de empresas particulares (Bernardo, 2012, pp. 25 - 41).

Para administrar eficientemente los sitios complejos se emplea un software que permite el desarrollo de aplicaciones web robustas e interactuar con las bases de datos, crear mapas de imágenes, multimedia, formularios, como se aprecia en el video 33.

Video 33. Introducción al diseño de páginas web.

Al mismo tiempo otras empresas como Adobe desarrollaron software para crear animaciones como Flash y Dreamweaver para crear sitios. La gran ventaja de Dreamweaver es que no tiene que ser un experto en el manejo del lenguaje HTML para producir y desarrollar sitios web de calidad (Adobe, 2019).

Otra empresa para la creación de sitios web es Sun Microsystems su principal aportación es el lenguaje java. La evolución de las computadoras ha permitido que muchos de los desarrollos o aplicaciones puedan ser empleados en diversos dispositivos móviles, las dos plataformas de desarrollo se distinguen en el video 34.
